# bike-sales-
Bike Sales Dashboard (Excel Project)  Developed an interactive Bike Sales Dashboard in Microsoft Excel using Pivot Tables, Pivot Charts, and data cleaning techniques to analyze customer purchase behavior.
Key Insights

* Compared average income by gender and purchase decision
* Analyzed customer age brackets and bike purchase trends
* Studied commute distance impact on bike purchases
* Built a clean visual dashboard for quick decision-making

Tools Used

Microsoft Excel, Pivot Table, Pivot Chart, Data Cleaning, Dashboard Design
